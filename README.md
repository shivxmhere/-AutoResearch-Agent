# AgentVerse
### The internet of agents — where every task finds its mind.

🚀 **Live Demo:** https://frontend-puce-nu-66.vercel.app
⚙️ **API:** https://agentverse-backend.vercel.app

AgentVerse is a full-stack AI agents marketplace where developers publish autonomous AI agents 
and users deploy them for any task — in one click, with zero setup.

## Live Agents

| Agent | Category | Speed | Accuracy | Cost |
|-------|----------|-------|----------|------|
| AutoResearch | Research | 90s | 87% | $0.00 |
| CompeteScope | Intelligence | 60s | 83% | $0.00 |
| CodeReview | Code | 45s | 91% | $0.00 |

## Key Features
- **Task-First Search** — describe your problem, get matched to the right agent instantly
- **Agent DNA Cards** — structured capability fingerprints for every agent (speed, accuracy, cost, tools)
- **Agent Arena** — run two agents on the same task, watch them race side by side in real-time
- **SSE Streaming** — watch each sub-agent's thought process live as it happens
- **Creator Economy** — usage stats, star ratings, run counts per agent

## Stack
- **Backend:** Python 3.11, FastAPI, LangGraph, LangChain, Groq (LLaMA 3.3 70B)
- **Frontend:** Next.js 14, TypeScript, Tailwind CSS v4, Framer Motion
- **AI:** Groq LPU (300 tok/s), Tavily Search, Serper API, FAISS Vector DB
- **Infra:** Vercel (frontend), free-tier APIs — $0.00 cost at demo scale
